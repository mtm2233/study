# typeScript

## 1.进入`ts-node` 或 `ts-webpack` 目录

## 2.安装依赖
```
npm i
```

## 3.运行
### `ts-node` 目录

### 编译执行单个 `.ts` 文件
- main.ts
  - ``npm run serve``
  - ``npx ts-node main.ts``

- 其他ts文件
  - ``npx ts-node xxx.ts``


### `ts-webpack` 目录

### 打包
- ``npm run build``

### 启动服务（编译执行.ts文件，监听.ts变化）
- ``npm run serve``
